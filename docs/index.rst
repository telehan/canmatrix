Welcome to canmatrix's documentation!
=====================================
*Canmatrix* implements a *Python Can Matrix Object* which describes the can-communication and the needed objects (Boardunits, Frames, Signals, Values, ...)
*Canmatrix* also includes two **Command line tools** (*canconvert* and *cancompare*) for converting and comparing **CAN** databases.
